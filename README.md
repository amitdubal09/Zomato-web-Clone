🍔 Zomato Clone Website

This is a front-end Zomato clone website built using HTML, CSS, JavaScript, and GSAP. The project focuses on replicating the modern UI of a food delivery platform with smooth animations and responsive design.

🚀 Features
🔥 Modern and clean UI design
📱 Fully responsive layout
🎯 Smooth animations using GSAP
🍽️ Food listing and sections similar to Zomato
🧭 Easy navigation and user-friendly interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
GSAP (GreenSock Animation Platform)

📂 Project Type
This is a front-end only project, meaning it does not include backend functionality like login, database, or real food ordering.
